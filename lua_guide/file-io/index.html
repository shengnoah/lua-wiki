Lua 文件 I/O

Lua 的 IO 库用于读取或操作文件。Lua IO 库提供两类文件操作，它们分别是隐式文件描述符(implict file descriptors)和显式文件描述符(explicit file descriptors)。

在接下来的例子的，我们会用到一个示例文件 test.lua，文件内容如下：

file = io.open (filename [, mode])

– 只读模式打开文件
file = io.open(“test.lua”, “r”)

– 将 test.lua 设置为默认输入文件
io.input(file)

–打印输出文件的第一行
print(io.read())

– 关闭打开的文件
io.close(file)

– 以追加模式打开文件
file = io.open(“test.lua”, “a”)

– 将 test.lua 设置为默认的输出文件
io.output(file)

– 将内容追加到文件最后一行
io.write(“– End of the test.lua file”)

– 关闭打开的文件
io.close(file)

  • Sample test.lua – 只读模式打开文件
    file = io.open(“test.lua”, “r”)

– 输出文件的第一行
print(file:read())

– 关闭打开的文件
file:close()

– 以追加模式打开文件
file = io.open(“test.lua”, “a”)

– 添加内容到文件的尾行
file:write(“–test”)

– 关闭打开的文件
file:close()

– Opens a file in read
file = io.open(“test.lua”, “r”)

file:seek(“end”,-25)
print(file:read(“*a”))

– closes the opened file
file:close()
